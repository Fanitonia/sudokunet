## About

A library for working with Sudoku puzzles. It provides tools for generating, solving, and validating Sudoku grids with ease.

## Getting Started

### Prerequisites

- .NET 8.0

### Installation

1. Add package to your project
   ```sh
   dotnet add package SudokuNet
   ```
